# Algebra / Equations — Asperger Tutor Reference

## Instructional Sequence (Without Skipping Steps)
1. Equality → **balance scale model**: left $=$ right; whatever operation is performed on one side must be performed on the other.
2. Unknown → "a number whose value is unknown" (not an abstract symbol): $2 + \square = 5$.
3. Single-variable equations → two variables → systems of equations.
4. Linear → quadratic → polynomial.

## Critical Concretization Points

**What is a variable?**
Do not do: "$x$ is a symbol representing the unknown."
Do do: "The number inside the box. $2 + \square = 5 \Rightarrow \square = 3$. We write $x$ instead of the box: $2 + x = 5$."

**Adding negative numbers:**
Plain-text number line (left = negative):
`... -3  -2  -1   0   1   2   3 ...`
$5 + (-3)$: Move $3$ steps to the left from $5$ $\Rightarrow 2$.

**Solving equations (balance scale model):** same operation to both sides, one operation per line:
$$2x + 3 = 11$$
$$2x = 8 \qquad (\text{subtract } 3 \text{ from both sides})$$
$$x = 4 \qquad (\text{divide both sides by } 2)$$

**Factoring:** reverse direction first — multiply, expand, then ask "where did it come from?":
$$(x + 2)(x + 3) = x^2 + 5x + 6$$
Then reverse it: show how to break down $x^2 + 5x + 6$ into $(x+2)(x+3)$.

## Common Working Memory Errors

| Error | Root Cause | Checkpoint |
|---|---|---|
| Sign error | Too many steps at once | One operation per line; explicitly write the sign when moving terms |
| Omitting parentheses | Visual fragmentation | Set up the parentheses first, then execute the operation |
| Moving to the wrong side | Rule remained abstract | Return to the balance scale model: "same to both sides" |
| Skipping distribution | Too many terms | Write the rule-card: $a(b+c) = ab + ac$ |

## Format: Equation Solving
**GIVEN:** [equation, LaTeX]
**STEP 1:** [isolate the constant]
**STEP 2:** [divide by the coefficient]
**RESULT:** $x = \dots$
**✔ CHECK:** [substitute the value back in — is the equality satisfied?]
