# Arithmetic / Number — Asperger Tutor Reference

## Teaching Sequence
1. Number = quantity (concrete object / number line).
2. Four operations → pair each with its inverse operation ($+ \leftrightarrow -$, $\times \leftrightarrow \div$).
3. Fraction = divided whole → decimal → percentage (three notations of the same thing).
4. Order of operations (system rule).
5. Divisibility / prime number / GCD-LCM.

## Critical Concrete Concepts

**What is a fraction?** "Part of a whole" — with visuals / objects:
$$\frac{3}{4} = \text{3 out of 4 equal parts}$$
Why is a common denominator needed? Parts of different sizes cannot be added directly. $\frac{1}{2} + \frac{1}{3}$ → convert both into parts of $6$:
$$\frac{1}{2} + \frac{1}{3} = \frac{3}{6} + \frac{2}{6} = \frac{5}{6}$$

**Three notations of the same thing (make the connection):**
$$\frac{1}{2} = 0.5 = 50\%$$
Not a new concept — just a change of format.

**Order of operations** (not arbitrary, a system rule): first parentheses, then exponents, then $\times \div$, last $+\,-$:
$$2 + 3 \times 4 = 2 + 12 = 14 \qquad (14, \; \text{not } 20)$$

**Negative number:** direction on the number line. Inverting twice: $-(-3) = +3$.

## Common Working-Memory Errors

| Error | Why it happens | Checkpoint |
|---|---|---|
| Skipping priority | Left-to-right automatism | Rank the operations first, then calculate |
| Adding without a common denominator | Fraction is viewed as an integer | Ask: "Are the part sizes the same?" |
| Decimal ↔ percentage confusion | The three notations are thought to be separate | $\times 100 \Rightarrow \%$, $\div 100 \Rightarrow$ decimal card |
| Losing the negative sign | Too many steps at once | Explicitly carry the sign over to every line |

## Format: Multi-Step Calculation
**GIVEN:** [expression, LaTeX]
**PRIORITY:** [which operation comes first — mark it]
**STEP 1-3:** [each line is a single operation]
**RESULT:** [value]
**✔ CHECK:** [magnitude estimation — is the result reasonable?]
